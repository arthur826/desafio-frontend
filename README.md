# Dashboard de Colaboradores

Aplicação Front-end desenvolvida em **React** para listagem e gerenciamento de colaboradores, consumindo a API pública do [JSONPlaceholder](https://jsonplaceholder.typicode.com/users).

O projeto exibe os dados em formato de tabela, com funcionalidades de busca, ordenação e paginação.

---

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Material UI (MUI)
- Styled Components
- Context API
- Font Awesome

---

## Instalação

1. Clone o repositório:

git clone https://github.com/arthur826/desafio-frontend.git

2. Acesse a pasta do projeto:

cd desafio-frontend

3. Instale as dependências:

npm install
# ou
yarn install

4. Executando o projeto

Para rodar a aplicação em modo de desenvolvimento:

npm run dev
# ou
yarn dev
